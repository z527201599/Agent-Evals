
# Agent 类型
- 你的Agent是做什么的？是任务导向型 (如客服、办公自动化)、对话型 (如社交陪伴)、创作型 (如写作、绘画)、决策型 (如游戏、模拟)、还是通用型？

# 核心能力
- 定义核心能力：任务完成能力、信息处理能力、语言能力（理解）、规划与决策、工具使用、效率、安全与伦理、用户交互体验

# 定义指标与评分标准
- 任务成功率：任务是否被正确完成？（完成度百分比）
- 精确度/召回率
- 代码通过率
- 步骤效率：完成任务所需的平均步骤数、规划步数
- 响应延迟
- 成本（重要）：执行任务消耗的计算资源 （API Token 数、GPU 时间等）

# 增加人工主观评估
- 相关性：输出是否紧扣需求
- 信息准确性、连贯性与流畅度
- 有帮助性（重要）：能否真正提供了有价值的信息？
- 创造性
- 安全性、伦理符合度
- 拟人化、用户体验

# 打分标准
- 为每个指标定义评分等级 （如 1-5 分）（通过/部分通过/未通过）
